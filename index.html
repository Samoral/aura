What is a signal

Signal is an abstraction of any measurable quantity represented as a function of an independent variables such as space, time, etc.

Signals are written in the form x(t), where:

x = Signal value
t = Independent variable

Hence, for a quantity to be a signal, it must vary (change) with respect to an independent variable.

FOR EXAMPLE

Alternating Current is a signal because the current (which is the quantity) changes with time.

Direct Current is not a signal because the current (which is the quantity) is constant at every point in time.

Systems

System is an abstraction of anything (e.g devices) that takes an input signal, operates on it and produces an output signal

Types Of System


Continous Time System:- It operates on continous time signal

Discrete Time System:- It operates on discrete time signal

Linear Time Invariance (LTI) System:- It combines the properties of linearity and time invariance


Let's talk about LTI systems..

LTI Systems

LTI systems are systems that combines the properties of linearity and time invariance.

Linearity in the sense that it allows an additional signal to flow in at the input, and produces an additional signal at the output.

Odd Signals

A signal is odd if the time reversed signal, x(-t) is equal to the negative form of the original signal, x(t), thereby making it anti-symmetrical about the y axis. i.e x(-t) = -x(t)

Let's take some examples

Examples

1. Is the signal below odd?

Solution

Multiply the time scale by -1 to find the time reversed version, x(-t)

at time, t = 0, the reversed time becomes 0 × -1 = 0

at time, t = 5, the reversed time becomes 5 × -1 = -5

at time, t = 10, the reversed time becomes 10 × -1 = -10

Now, redrawing the signal gives us x(-t) as

Since the time reversed signal, x(-t) is the -ve form of the original signal, x(t). Hence, the signal is odd.

Note: All odd signal must be zero at t=0.

You might wonder why all odd signals are zero when t = 0..


Let's Take Some Examples

1.
Say, x(t) = t

Solution:-

Remember for a signal to be odd, it has to satisfy the equation x(-t) = -x(t)

Given, x(t) = t. Find x(-t)

Then x(-t) = x(t) × μ

Substituting t for x(t) and -1 for μ

x(-t) = t × -1 = -t

Substituting in odd signals equation

x(-t) = -x(t)
i.e,
-t = -x(t) = -t (odd)

Checking if x(t) is zero at t = 0..

-t = -t = x(t)

x(0) = -(0) = 0 (proved!)

If you try for multiple odd values, it is satisfied that all odd signals, x(t) is zero when t = 0

Important notes on Odd and Even signals


1. Products of two even signals is an even signal

Proof..

let us have two even signals, x1(t) and x2(t).

let p(t) = products of the two even signals

p(t) = x1(t)•x2(t) --- (i)

Performing time reversal

p(-t) = x1(-t)•x2(-t) --- (ii)

Recall: x(t) = x(-t) [if even]

Therefore, x1(t) = x1(-t) and x2(t) = x2(-t)

Substituting in equation (i)..

p(-t) = x1(t)•x2(t)

But, from eqn. (i) p(t) = x1(t)•x2(t)

Then, p(-t) = p(t) [even].. Proved!

2. Products of two odd signals is an even signal

Proof..

let us have two odd signals, x1(t) and x2(t).

let p(t) = products of two odd signals

p(t) = x1(t)• x2(t) --- (i)

Performing time reversal

Recall: x(-t) = -x(t) [for odd signals]

p(-t) = -x1(t) • -x2(t)
p(-t) = x1(t) • x2(t)--- (ii)

But, p(t) = x1(t) • x2(t)

•°• p(-t) = p(t) [even] Proved…!

3. Products of one odd and one even signal is an odd signal

Proof..

let x1(t) be odd signal and x2(t) be even signal.

let p(t) = products of two odd signals

p(t) = x1(t)• x2(t) --- (i)

Performing time reversal

p(-t) = x1(-t)• x2(-t) --- (ii)

Recall: x(-t) = -x(t) [for odd signals], then
x1(-t) = -x1(t)

x(-t) = x(t) [for even signals] , then
x2(-t) = x2(t)

substituting in eqn. (ii)

p(-t) = -x1(t) • x2(t)


But, p(t) = x1(t) • x2(t)

•°• p(-t) = -p(t) [odd] Proved…!

9. Conjugate Symmetric Signal:-

The original signal, x(t) is the same as the complex conjugate of the time reversed version, x*(t) of the original signal. i.e x(t) = x*(-t)

x*(-t) = Conjugate Signal

Example

1. Check if the signal is conjugate symmetrical. x(t) = x1(t) + jx2(t).

Solution
Perform time reversal (folding operation)
x(-t) = x1(t) + x2(t).
Take the conjugate
x*(-t) = x1(-t) + jx2(-t).

Since, x(t) = x*(-t) then, it is conjugate signal.

10. Conjugate Anti-symmetric Signal:-

x(t) = -x*(-t)

11. Periodic Signal:- A signal is periodic when there is a positive non-zero value at time period. It is given as

x(t) = x(t+T) £ t
•°• if T is a positive non-zero value (e.g 1,2,3..∞), then it is periodic.

Recall: T = ¹/f. Then, the smallest positive value of T is given as T• = ¹/f•

Note:- All Continous time signals are periodic but Discrete time signals are only positive if there is a positive integer n to satisfy the equation x(n = x(n + N) £ n

Remember a signal is periodic if we have a positive value of T or N to satisfy the above equations. Lets talk about the next signal..

12. Energy Signal

A signal, x(t) is energy signal if and only if the total energy is finite [i.e it is not infinite] and is integrable.

Equation..

$$0 < E < ∞; P_{avg} = 0$$

$$E = \int_{-\infty}^{\infty} {|x(t)|}^2 \, dt < ∞$$

Examples of energy signals

  1. Rectangular pulse
  2. Triangular pulse
  3. Exponential decay signal

12. Power Signal

If the average power is finite and non-zero, the signal is a power signal.

Equation..

$$0 < P < ∞; E = ∞$$

$$P_{avg} = \lim_{T\to\infty}\frac{1}{T} \int_{\frac{-T}{2}}^{\frac{T}{2}}x(t)^2dt$$

Examples of power signals

  1. Sine function
  2. Cosine function
  3. Periodic signals

$$E = \left[ \frac{e^{-at}}{-2a} \right]_{0}^{\infty} $$

$$E = \left[ \frac{e^{-a(\infty)}}{-2a} \right]_{0}^{\infty} - \left[ \frac{e^{-a(0)}}{-2a} \right]_{0}^{\infty} $$

$$E = \left[ \frac{0}{-2a} \right] - \left[ \frac{1}{-2a} \right] $$

$$E = \frac{1}{2a}$$ taking a as any value greater than 0, for example 1, we get 0.5 which is \lt∞. i.e, it is a finite value

Since, the value is finite. It is therfore an energy signal.
